ElementUI-input的type='number'时去掉上下箭头和鼠标滚轮滚动事件

发表于 2021-04-01 更新于 2021-05-24 分类于 前端 阅读次数：
ElementUI-input的type='number'时去掉上下箭头和鼠标滚轮滚动事件的小技巧，在实际开发中可以使用，亲测有效。

话不多说，直接贴代码，亲测有用~

html代码

  • 用去禁用input输入框中滚轮可以调节数字的事件

    • <el-input
    type="number" // input的type为"number"
    min="0"
    style="flex: 1; text-align: center;"
    v-model="Form"
    autocomplete="off"
    @mousewheel.native.prevent  // 添加上这句代码即可，禁用鼠标滚动事件
>
</el-input>

css代码

  • 用于去除input中的上下箭头：

    • input::-webkit-outer-spin-button,
input::-webkit-inner-spin-button {
  -webkit-appearance: none !important;
}
input[type='number'] {
  -moz-appearance: textfield;
}

  • 如果以上代码没有作用，查看style标签是否添加了scoped，如果是，那么需要在进行样式穿透，即在样式代码块前添加::v-deep或者/deep/即可

    • /deep/ input::-webkit-outer-spin-button,
/deep/ input::-webkit-inner-spin-button {
  -webkit-appearance: none !important;
}
/deep/ input[type='number'] {
  -moz-appearance: textfield;
}
